# Web Calculator

## Description

Simple web calculator for arithmetic expressions. 
Supports basic arithmetic operations and has a beautiful web presentation. 🤖✨

## Tech Stack

- Spring Boot 🍃
- Spring MVC
- Thymeleaf
- JUnit ✅

## Features

- Calculates expressions with integers and decimals 🧮
- Input validation (checks for invalid expressions, division by zero) 🛡️
- Converts expressions to Reverse Polish Notation (RPN) ⚙️
- Web interface + REST API 🌐

## Testing

JUnit tests cover:
- Expression validation 🧪
- Calculation correctness ✅
- Error handling ⚠️


🚀 Happy calculating!
